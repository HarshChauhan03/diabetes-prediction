# 🩺 Diabetes Prediction

## 📌 Overview

This project predicts whether a patient is likely to have diabetes based on health parameters such as glucose level, blood pressure, and age.

The project demonstrates a healthcare machine learning classification workflow.

---

## 🎯 Objective

- Predict diabetes risk
- Learn healthcare AI applications
- Understand classification algorithms

---

## 🧠 Workflow

```text
Patient Data
↓
Data Preprocessing
↓
Logistic Regression
↓
Diabetes Prediction
```
